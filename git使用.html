一般流程
第一次下载某个远程库时
新建一个文件夹
右键git bash 然后init一下 然后 clone远程库
成功后可以看到远程库文件夹
再右键该文件夹git bash 然后init
此时再对整个项目做出修改
然后 add -A
git commit-m''
然后 push就可以了。
在第二次使用这个本地库文件夹是，由于远程库中文件与本地库文件不一致，所以 先pull --rebase orinjin master 合并本地和远程库

或者，直接把本地库删掉，重新clone


1.创建本地仓库 git init
2.添加远程仓库地址 git remote add [name] [ssh地址或者HTTPs地址]
3.添加或修改文件 git add -A
4.git commit 
4.git push -u [name] master
